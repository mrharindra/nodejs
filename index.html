User

  • Signup: POST {HOST_NAME}/signup
  • Login: POST https://ngtodoapp.herokuapp.com/login


  • Get all users: GET https://ngtodoapp.herokuapp.com/user
  • Get user by ID: GET https://ngtodoapp.herokuapp.com/user/{id}
  • Update User PUT https://ngtodoapp.herokuapp.com/user/{id}
  • Delete user: DELETE https://ngtodoapp.herokuapp.com/user/{id}
    • User schema 
     
            { 
                "name":"",
                "email":"",
                "password":""
            }
